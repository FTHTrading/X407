# Security Policy

## Reporting

If you identify a vulnerability, infrastructure weakness, abuse vector, or data exposure risk related to X407 materials or connected systems, do not disclose it publicly first.

Report security issues privately to **security@fth.trading** or via the [GitHub Security Advisories](https://github.com/FTHTrading/X407/security/advisories) tab.

## Preferred report contents

- affected component
- reproduction steps
- severity estimate
- proof of concept, if available
- impact assessment
- recommended mitigation, if known

## Scope

This repository may include strategic architecture, compliance workflows, and commercialization material. Some operational systems and protected services may exist outside the public repository.

## Disclosure expectations

Responsible disclosure is expected. Public disclosure should wait until remediation is completed or authorized.
